Download Avocado

Get yourself some fresh Avocado

Avocado code can be found at https://github.com/avocado-framework/avocado/
For detailed instructions and how to install from source, please check our documentation.


Python package on PyPI
You can install the package from PyPI: 

$ pip3 install avocado-framework
More information at https://pypi.org/project/avocado-framework/

Also, check out for the avocado plugins distributed on PyPI https://pypi.org/search/?q=avocado-framework-plugin


Module for Fedora
You only need to enable the module avocado:latest and you'll get avocado updates in the future: 

# dnf module enable avocado:latest
# dnf module install avocado


YUM repository for Enterprise Linux
Get the repository file from https://avocado-project.org/data/repos/avocado-el.repo
